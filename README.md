# QBKO UI

A demo Expo app for studying and experimenting with high-quality UI components targeting gamified applications. Components are built with Reanimated, Skia, and Rive to achieve smooth animated behavior that is both visually polished and production-ready.

## Tech Stack

- **Expo SDK 54** (managed workflow with prebuild)
- **React Native 0.81** + **TypeScript** (strict mode)
- **@rive-app/react-native** for Rive integration
- **expo-router** for navigation
- **react-native-reanimated** for animations
- **@shopify/react-native-skia** for Skia rendering
- **EAS Build** for iOS and Android builds
